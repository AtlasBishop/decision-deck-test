# Decision Deck - OnMission

A comprehensive readiness and alignment diagnostic tool built with React and Tailwind CSS.

## Features

- **Dashboard View**: Visual overview with OnMission Score, priority actions, layer completion, and readiness heatmap
- **Detail View**: Deep dive into each layer with expandable requirements, FOG diagnostic, and Key Practice frameworks
- **Timeline View**: Sequential implementation roadmap across all 5 layers
- **Decision Logic View**: AI support rules and timing guidelines based on OnMission Score
- **AI Insights View**: Different insights for executives vs. frontline workers

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Opens at [http://localhost:3000](http://localhost:3000)

### Build

```bash
npm run build
```

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Lucide React Icons
